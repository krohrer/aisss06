Project Proposal

What do we want to do?

We would like to implement a program that is able to "grow" buildings, gradually refining the visual representation as the viewer gets closer. For this, we would like to model everything as geometry instead of just applying a `facade'-texture to a rough geometric approximation of the building like in [1].

What results do we want to show?

Rendering procedurally generated buildings in real-time, with some kind of LoD.

What are the technical problems we have to solve?

Investigate different approaches to generate/grow buildings.
Implement view-dependent LoD.

Which steps are necessary to solve those problems?

Read up on ways to generate gemetry.
Find a suitable representation for the geometry, possibly CSG.
Try out different implementations...

...debugging

What third-party software or libraries do we want to use?

OpenGL, GLUT

What software parts do we implement ourselves?

Generation and rendering of the buildings
