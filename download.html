Simply follow the instructions in README.TXT in the official github repository.

Prerequisites

You will need an implementation of the Ocaml programming language. And you will also need the lablgl library to use OpenGL and Glut with with Ocaml. On some systems, both packages can be retrieved using the package manager. This is the easiest way to install the prerequisites.

Sourcecode

After downloading the sourcecode, extract the archive with `tar -xvzf ...`. Then follow the directions of the README.TXT to build and run the application.
